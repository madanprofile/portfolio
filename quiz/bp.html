BOOTSTRAP QUIZ

Q 1. Bootstrap 3 is mobile-first.

Ans:- True

Q 2. Which class provides a responsive fixed width container?

Ans:- .container

Q 3. Which class provides a full width container, spanning the entire width of the viewport?

Ans:- .container-fluid

Q 4. The Bootstrap grid system is based on how many columns?

Ans:- 12

Q 5. Which class adds zebra-stripes to a table?

Ans:- .table-striped

Q 6. Which class shapes an image to a circle?

Ans:- .img-circle

Q 7. Which class is used to create a big box for calling extra attention?

Ans:- .jumbotron

Q 8. Which button class is used to create a large button?

Ans:- .btn-lg

Q 9. Which class is used to create a button group?

Ans:- .btn-group

Q 10. How can you insert a search icon?

Ans:- <span class="glyphicon glyphicon-search">

Q 11. Which class is used to create a badge?

Ans:- .badge

Q 12. Which class is used to create a basic pagination?

Ans:- .pagination

Q 13. Which class is used to create a basic list group?

Ans:- .list-group

Q 14. Which class adds a heading to a panel?

Ans:- .panel-heading

Q 15. Which class indicates a dropdown menu?

Ans:- .dropdown

Q 16. A standard navigation tab is created with:

Ans:- <ul class="nav nav-tabs">

Q 17. A standard navigation bar is created with:

Ans:- <nav class="navbar navbar-default">

Q 18. Which class is used to create a black navigation bar?

Ans:- .navbar-inverse

Q 19. Which plugin is used to cycle through elements, like a slideshow?

Ans:- Carousel

Q 20. Which plugin is used to create a modal window?

Ans:- Modal

Q 21. Which plugin is used to create a tooltip?

Ans:- Tooltip

Q 22. Which contextual class indicates a succesful or positive action?

Ans:- .text-success

Q 23. Which contextual class indicates a dangerous or potentially negative action?

Ans:- .text-danger

Q 24. Which class indicates uppercased text?

Ans:- .text-uppercase

Q 25. The Bootstrap grid system works across multiple devices.

Ans:- True
