HTML QUIZ

Q 1. What does HTML stand for?

Ans:- Ans:- Hyper Text Markup Language

Q 2. Who is making the Web standards?

Ans:- The World Wide Web Consortium

Q 3. Choose the correct HTML element for the largest heading:

Ans:- <h1>

Q 4. What is the correct HTML element for inserting a line break?

Ans:- <br>

Q 5. What is the correct HTML for adding a background color?

Ans:- <body style="background-color:yellow;">

Q 6. Choose the correct HTML element to define important text

Ans:- <strong>

Q 7. Choose the correct HTML element to define emphasized text

Ans:- <em>

Q 8. What is the correct HTML for creating a hyperlink?

Ans:- <a href="http://www.mmsonu.6te.net.com">mmsonu</a>

Q 9. Which character is used to indicate an end tag?

Ans:- /

Q 10. How can you open a link in a new tab/browser window?

Ans:- <a href="url" target="_blank">

Q 11. Which of these elements are all <table> elements?

Ans:- <table><tr><td>

Q 12. Inline elements are normally displayed without starting a new line.

Ans:- True

Q 13. How can you make a numbered list?

Ans:- <ol>

Q 14. How can you make a bulleted list?

Ans:- <ul>

Q 15. What is the correct HTML for making a checkbox?

Ans:- <input type="checkbox">

Q 16. What is the correct HTML for making a text input field?

Ans:- <input type="text">

Q 17. What is the correct HTML for making a drop-down list?

Ans:- <select>

Q 18. What is the correct HTML for making a text area?

Ans:- <textarea>

Q 19. What is the correct HTML for inserting an image?

Ans:- <img src="image.gif" alt="MyImage">

Q 20. What is the correct HTML for inserting a background image?

Ans:- <body style="background-image:url(background.gif)">

Q 21. <iframe> is used to display a web page within a web page.

Ans:- True

Q 22. HTML comments start with <!-- and end with -->

Ans:- True

Q 23. Block elements are normally displayed without starting a new line.

Ans:- False

Q 24. Which HTML element defines the title of a document?

Ans:- <title>

Q 25. Which HTML attribute specifies an alternate text for an image, if the image cannot be displayed?

Ans:- alt

Q 26. Which doctype is correct for HTML5?

Ans:- <!DOCTYPE html>

Q 27. Which HTML element is used to specify a footer for a document or section?

Ans:- <footer>

Q 28. In HTML, you can embed SVG elements directly into an HTML page.

Ans:- True

Q 29. What is the correct HTML element for playing video files?

Ans:- <video>

Q 30. What is the correct HTML element for playing audio files?

Ans:- <audio>

Q 31. The HTML global attribute, "contenteditable" is used to:

Ans:- Specify whether the content of an element should be editable or not

Q 32. In HTML, onblur and onfocus are:

Ans:- Event attributes

Q 33. Graphics defined by SVG is in which format?

Ans:- XML

Q 34. The HTML <canvas> element is used to:

Ans:- draw graphics

Q 35. In HTML, which attribute is used to specify that an input field must be filled out?

Ans:- required

Q 36. Which input type defines a slider control?

Ans:- range

Q 37. Which HTML element is used to display a scalar measurement within a range?

Ans:- <meter>

Q 38. Which HTML element defines navigation links?

Ans:- <nav>

Q 39. In HTML, what does the <aside> element define?

Ans:- Content aside from the page content

Q 40. Which HTML element is used to specify a header for a document or section?

Ans:- <header>
