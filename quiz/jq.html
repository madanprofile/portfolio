jQuery QUIZ

Q 1. Which of the following is correct?

Ans:- jQuery is a JavaScript Library

Q 2. jQuery uses CSS selectors to select elements?

Ans:- True

Q 3. Which sign does jQuery use as a shortcut for jQuery?

Ans:- the $ sign

Q 4. Look at the following selector: $("div"). What does it select?

Ans:- All div elements

Q 5. Is jQuery a library for client scripting or server scripting?

Ans:- Client scripting

Q 6. Is it possible to use jQuery together with AJAX?

Ans:- Yes

Q 7. The jQuery html() method works for both HTML and XML documents

Ans:- False

Q 8. What is the correct jQuery code to set the background color of all p elements to red?

Ans:- $("p").css("background-color","red");

Q 9. With jQuery, look at the following selector: $("div.intro"). What does it select?

Ans:- All div elements with class="intro"

Q 10. Which jQuery method is used to hide selected elements?

Ans:- hide()

Q 11. Which jQuery method is used to set one or more style properties for selected elements?

Ans:- css()

Q 12. Which jQuery method is used to perform an asynchronous HTTP request?

Ans:- jQuery.ajax()

Q 13. What is the correct jQuery code for making all div elements 100 pixels high?

Ans:- $("div").height(100)

Q 14. Which statement is true?

Ans:- To use jQuery, you can refer to a hosted jQuery library at Google

Q 15. What scripting language is jQuery written in?

Ans:- JavaScript

Q 16. Which jQuery function is used to prevent code from running, before the document is finished loading?

Ans:- $(document).ready()

Q 17. Which jQuery method should be used to deal with name conflicts?

Ans:- noConflict()

Q 18. Which jQuery method is used to switch between adding/removing one or more classes (for CSS) from selected elements?

Ans:- toggleClass()>

Q 19. Look at the following selector: $("div p"). What does it select?

Ans:- All p elements inside a div element

Q 20. Is jQuery a W3C standard?

Ans:- No

Q 21. Look at the following selector: $("p#intro"). What does it select?

Ans:- The p element with id="intro"

Q 22. Which jQuery method is used to remove selected elements?

Ans:- Both methods can be used

Q 23. Look at the following selector: $(":disabled"). What does it select?

Ans:- All disabled input elements

Q 24. Which jQuery method returns the direct parent element of the selected element?

Ans:- parent()

Q 25. The jQuery animate() method can be used to animate ANY CSS property?

Ans:- Only properties containing numeric values
